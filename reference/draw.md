# draw Interactive Map

draw Interactive Map

## Usage

``` r
draw(
  isoData,
  zoom = 5,
  type = "1",
  northArrowPosition = "none",
  scalePosition = "none",
  center = NULL,
  bounds = NULL
)
```

## Arguments

- isoData:

  isoData data

- zoom:

  zoom

- type:

  map type

- northArrowPosition:

  position of north arrow

- scalePosition:

  position of scale

- center:

  where to center map (list of lat and lng)

- bounds:

  map bounds (list of north, south, east, west)
