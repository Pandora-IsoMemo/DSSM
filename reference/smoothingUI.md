# Smoothing UI

Smoothing UI

## Usage

``` r
smoothingUI(
  ns,
  label_slider = "No. of basis functions",
  label_info = "Basis functions",
  min = 20,
  max = 1000,
  value = 70,
  step = 10
)
```

## Arguments

- ns:

  namespace

- label_slider:

  label of the slider

- label_info:

  label of the info button

- min:

  minimum value of the slider

- max:

  maximum value of the slider

- value:

  default value of the slider

- step:

  step of the slider
