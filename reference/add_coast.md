# Add coastline layer (generic)

Generic for adding a coastline layer to a map plot.

## Usage

``` r
add_coast(x, ...)

# S3 method for class 'MapLayers'
add_coast(x, ...)
```

## Arguments

- x:

  A `MapLayers` object.

- ...:

  Further arguments passed to methods.

## Details

For `MapLayers`, `add_coast()` draws coastlines.
