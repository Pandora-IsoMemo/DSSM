# Numeric Input Lat And Long Server

Server function of the numeric input of latitude and longitude module

## Usage

``` r
numericInputLatAndLongServer(
  id,
  valueLat = reactive(50),
  valueLong = reactive(10)
)
```

## Arguments

- id:

  id of module

- valueLat:

  (numeric) latitude

- valueLong:

  (numeric) longitude
