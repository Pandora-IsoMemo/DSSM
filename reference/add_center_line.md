# Add center line (generic)

Generic for adding a center meridian line to a map plot.

## Usage

``` r
add_center_line(x, ...)

# S3 method for class 'MapLayers'
add_center_line(x, ...)
```

## Arguments

- x:

  A `MapLayers` object.

- ...:

  Further arguments passed to methods.

## Details

For `MapLayers`, `add_center_line()` draws a vertical center meridian at
longitude 0 (plot coords).
