# Filter Time

Filter Time

## Usage

``` r
filterT(data, addU, time)
```

## Arguments

- data:

  (data.frame) data to be filtered by time containing columns "Date" and
  "Uncertainty"

- addU:

  (numeric) additional uncertainty to be added to time frame

- time:

  (numeric) time for filter

## Value

(data.frame) filtered data
