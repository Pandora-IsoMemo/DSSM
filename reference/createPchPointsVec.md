# Create PCH Points Vector

Create PCH Points Vector

## Usage

``` r
createPchPointsVec(
  pch = 16,
  width = 50,
  height = 50,
  bg = "transparent",
  col = "black",
  ...
)
```

## Arguments

- pch:

  plotting ‘character’, i.e., symbol to use. See graphics::points for
  details

- width:

  width in pixel

- height:

  height in pixel

- bg:

  initial background colour

- col:

  color code or name

- ...:

  Further graphical parameters that are passed to graphics::points()
