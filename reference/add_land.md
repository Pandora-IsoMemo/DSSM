# Add land layer (generic)

Generic for adding a land layer to a map plot.

## Usage

``` r
add_land(x, ...)

# S3 method for class 'MapLayers'
add_land(x, ...)
```

## Arguments

- x:

  A `MapLayers` object.

- ...:

  Further arguments passed to methods.

## Details

For `MapLayers`, `add_land()` fills land polygons.
