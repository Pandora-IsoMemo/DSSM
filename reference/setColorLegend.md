# Set Colour Legend

Set Colour Legend

## Usage

``` r
setColorLegend(map, showLegend, title, pal, values)
```

## Arguments

- map:

  leaflet map

- showLegend:

  logical show/hide legend

- title:

  legend title

- pal:

  colour palette

- values:

  possible values that can be mapped, e.g. isoData\$source
