# Slider And Input UI

UI of the Slider And Input module

## Usage

``` r
sliderAndNumericInputUI(id, label, min, max, value, step)
```

## Arguments

- id:

  id of module

- label:

  label

- min:

  (numeric) minumum

- max:

  (numeric) maximum

- value:

  (numeric) default value

- step:

  (numeric) step
