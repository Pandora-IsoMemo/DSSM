# Add centroid of clusters

Add centroid of clusters

## Usage

``` r
addCentroids(
  dataC,
  cluster,
  centers,
  type = c("centroid_spatial_cluster", "temporal_group_reference_point")
)
```

## Arguments

- dataC:

  Filtered data

- cluster:

  Cluster ids

- centers:

  Cluster centers

- type:

  Type of centroid

## Value

Data with centroids
