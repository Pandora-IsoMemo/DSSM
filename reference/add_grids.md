# Add grid lines layer (generic)

Generic for adding grid lines to a map plot.

## Usage

``` r
add_grids(x, ...)

# S3 method for class 'MapLayers'
add_grids(x, ...)
```

## Arguments

- x:

  A `MapLayers` object.

- ...:

  Further arguments passed to methods.

## Details

For `MapLayers`, `add_grids()` overlays graticule lines.
