# Set North Arrow on a Leaflet Map

Sets the North Arrow on a Leaflet map based on the properties of the
NorthArrow object.

## Usage

``` r
# S3 method for class 'NorthArrow'
set(obj, map, ...)
```

## Arguments

- obj:

  A NorthArrow object.

- map:

  A Leaflet map object.

- ...:

  Additional arguments (not used).

## Value

A Leaflet map object with the North Arrow added.
