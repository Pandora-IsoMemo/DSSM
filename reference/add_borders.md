# Add borders layer (generic)

Generic for adding political/country borders to a map plot.

## Usage

``` r
add_borders(x, ...)

# S3 method for class 'MapLayers'
add_borders(x, ...)
```

## Arguments

- x:

  A `MapLayers` object.

- ...:

  Further arguments passed to methods.

## Details

For `MapLayers`, `add_borders()` draws political borders.
